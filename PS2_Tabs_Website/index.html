EDI Projects

EDI Project 1: Smart Infant Cry Detection and Soothing System

This project focuses on developing an intelligent baby monitoring system using ESP32 microcontrollers, sensors, and machine learning. The system detects baby cries using a MAX4466 microphone, where audio signals are processed on the ESP32 and classified as cry or not cry using a TinyML model trained on Edge Impulse.

Along with cry detection, the system monitors baby movement, temperature, and humidity using ultrasonic and DHT11 sensors. When a cry is detected, real-time alerts are sent to a Firebase-backed mobile application, and a second ESP32 automatically plays soothing lullabies using an SD card and speaker module.

This project demonstrates effective use of embedded AI, sensor integration, cloud connectivity, and real-time decision-making for infant care automation.

EDI Project 2: Smart Crop Lifecycle Automation and AI-Based Soil NPK Estimation

This project presents a PC-based smart agriculture system that combines crop environment automation with AI-driven soil nutrient prediction. An Arduino Nano continuously collects environmental data such as air temperature, humidity, soil moisture, and soil temperature, which is transmitted to a Python-based supervisory application via serial communication.

The system controls actuators like fan, water pump, humidifier, and buzzer based on a four-stage crop lifecycle model, ensuring optimal growth conditions. Additionally, the project uses sensor fusion and computer vision, where soil images and pH paper images are processed and combined with live sensor data to predict Nitrogen (N), Phosphorus (P), and Potassium (K) values using a TensorFlow Lite model.

The solution offers a low-cost, fault-tolerant, and intelligent precision agriculture tool suitable for small-scale farmers and researchers.

Course Projects

Course Project 1: Air Gesture Mouse

The Air Gesture Mouse project focuses on developing a gesture-based human–computer interaction system that replaces traditional mouse and keyboard usage. The system uses an MPU6050 motion sensor to capture hand movements and an ESP32 microcontroller to process gesture data in real time.

Based on the detected gestures, cursor movement and keyboard actions are performed using Bluetooth or USB HID communication. A capacitive touch sensor is used to switch between mouse control mode and gesture-based keyboard control. This project helped in understanding embedded systems, sensor calibration, and real-time data processing.

Course Project 2: Cyber Surfy: AI Powered shoulder surfer detection system

This project explores facial landmark detection using the MediaPipe framework, specifically the Face Mesh model. The system detects and tracks facial landmarks in real time using a camera feed and computer vision techniques.main aim is to detect the surfer in public places like offices,garden.

The project involved understanding the difference between the MediaPipe framework and its Face Mesh solution, implementing facial feature tracking, and analyzing real-time performance. This work strengthened concepts related to computer vision, machine learning pipelines, and real-time image processing.
